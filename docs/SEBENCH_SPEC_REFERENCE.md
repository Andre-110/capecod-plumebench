# SE-Bench Spec Reference

The full SE-Bench task specification is stored in the benchmark workspace at:

```text
/root/SE-bench-main/tasks/capecod_plumebench.json
```

The full task blueprint directory is:

```text
/root/SE-bench-main/task_blueprints/capecod_plumebench
```

The public GitHub repository intentionally does not include the full judge
setup code, hidden target generation code, or hidden scorer internals. Those
belong in the private SE-Bench task package and judge image, not in the
agent-visible public task repository.
